Const RequireRole

  • RequireRole(roles: string | string[]): MethodDecorator
    • Defined in lib/decorators/RequireRole.ts:16

    Decorator to be used for access control. By default we suppose that all GraphQL resources are accessible to all users. If we want to limit the access to specific resource, we can add RequireRole decorator to resolver annotation. Valid roles are 'UNVERIFIED', 'LIMITED', 'USER', 'ADMIN', and 'MACHINE', you can also supply multiple roles that are allowed to access this resource by providing an array like ['ADMIN', 'MACHINE'].

    Parameters

    • roles: string | string[]

      role or roles needed to acces this resource
